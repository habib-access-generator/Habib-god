# HABIB ACCESS GENERATOR

🎯 A sleek, modern access token generator for game accounts with a cyberpunk-themed UI.

## Features

✨ **Modern Design** - Cyberpunk-themed interface with gradient animations
🔐 **Token Generation** - Extract and process access tokens securely
📋 **Easy Copy** - One-click copy functionality for tokens and JSON data
🎮 **Multi-Provider Support** - Google, Facebook, Apple, Twitter, VK authentication links
📱 **Responsive** - Works perfectly on desktop and mobile devices
🌐 **Social Integration** - Links to Telegram, YouTube, and Instagram

## How to Use

1. Click on your preferred social provider (Google, Facebook, etc.)
2. Complete the authentication process
3. Paste the URL or eat token in the input field
4. Click "GENERATE ACCESS" button
5. Copy your Account UID, Access Token, and Nickname

- **Tailwind CSS** - Styling
